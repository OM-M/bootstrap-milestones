Bootstrap Milestones bootstrap-milestones.css

A shiny bootstrap plugin for creating milestones. Shamelessly inspired by GitHub's Pull Request history.

  • How to get Bootstrap Milestones ?

  • Download bootstrap-milestones
    • From npm: npm install bootstrap-milestones
    • From bower: bower install bootstrap-milestones
    • From GitHub:
      git clone https://github.com/alexjab/bootstrap-milestones.git
    • Manually: minified version
  • Download bootstrap
    • You used npm or bower? You are good to go.
    • You cloned it ? Either run npm install or bower install from within the folder.
    • Something else ? Get bootstrap from here.
  • Reference both bootstrap and bootstrap-milestones in your project: 

    <link rel="stylesheet" href="bootstrap/dist/css/bootstrap.min.css">
<link rel="stylesheet" href="bootstrap-milestones/dist/bootstrap-milestones.min.css">

  • You're good too go !
